# PRD: ios-habit-app

## Problem

People trying to build daily habits often lose momentum because lightweight tracking apps either hide the daily action behind too many screens or push complex analytics before the check-in habit exists.

## Goal

Build a SwiftUI habit tracker example that makes the daily check-in fast, obvious, and testable.

## Users

- Individual tracking one to five habits.
- Student or builder learning SwiftUI project structure.

## MVP Scope

- Habit list.
- Add habit form.
- Daily completion toggle.
- Simple streak display.
- Local-only persistence boundary.

## Non-Goals

- Cloud sync.
- Social sharing.
- Reminders.
- Widgets.
- Subscription features.

## Acceptance Criteria

- User can create a habit.
- User can mark a habit complete for today.
- Streak display updates after completion.
- Tests cover habit creation and completion logic.
